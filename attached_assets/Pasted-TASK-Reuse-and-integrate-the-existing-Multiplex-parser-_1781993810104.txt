GOAL

Automatically generate Atmosfera cinema schedule every day at 06:00 Kyiv time and save it to Firestore so the Unity application can consume it.

FIRESTORE STRUCTURE

Cinema/
└ atmosfera/
└ cinema_schedule/
└ YYYY-MM-DD/
└ sessions/
└ SESSION_ID

Example:

Cinema/atmosfera/cinema_schedule/2026-06-21/sessions/98102

Document format:

{
"sessionId": "98102",
"movie": "Історія іграшок 5",
"hall": "Зал №1",
"startTime": "2026-06-21T10:00:00+03:00",
"endTime": "2026-06-21T11:42:00+03:00",
"generatedAt": firestore.SERVER_TIMESTAMP
}
